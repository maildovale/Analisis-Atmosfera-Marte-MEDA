# Manejo de Datos Meteorológicos desde Marte – MEDA (Perseverance)

Notebook en Python correspondiente a la Actividad 3 de la materia
Sistemas Planetarios y Astrobiología.

## Contenido
- Descarga y lectura de datos MEDA (PDS – NASA)
- Procesamiento temporal (ventanas de 5 minutos)
- Análisis de presión atmosférica
- Comparación estacional (4 soles representativos)
